Text animation


                        const text = 'My name is Karl Smith';
                        const typewriter1 = Array.from(text);
                        let header1 = document.querySelector('.header-border');
                        let header = document.querySelector('.header-name');
                        
                        function writeName1 () {
                        
                            for (let i = 0; i < typewriter1.length; i++) {
                        
                                setTimeout( () => {
                                    header1.innerHTML += typewriter1[i];
                                }, 150 * i);

                        });

SQL challenge


        
        SELECT 
            DISTINCT player_name AS "Player Name", 
            posi_to_play AS "Position", 
            jersey_no AS "Shirt Number", 
            player_id
        FROM player_mast
        WHERE player_id IN (
            SELECT player_id 
        FROM player_booked
            WHERE sent_off = 'Y'
        )
        ORDER BY player_id ASC
        ;

We were tasked with creating a query that included a sub query with a formatted output from a selection of databases. I chose to try and find who got sent off during the Euro 2016 football tournament. Included are the table structure, and my SQL query.

We used PostgreSQL for the challenge, and so i basically selected the distinct player names (since i only wanted to find out if they got sent off at any point, not using distinct would tell me if they got sent off more than once), and some other information i thought was important from the player_mast table. The subquery then checks the player_booked table, and returns all the player id's of the players who got sent off, and then the where condition matches the player id's returned, with player id's in the player_mast table to give me the information on everyone who got sent off.
