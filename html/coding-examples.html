SQL challenge


                        SELECT 
                            DISTINCT player_name AS "Player Name", 
                            posi_to_play AS "Position", 
                            jersey_no AS "Shirt Number", 
                            player_id
                        FROM player_mast
                        WHERE player_id IN (
                            SELECT player_id 
                        FROM player_booked
                            WHERE sent_off = 'Y'
                        )
                        ORDER BY player_id ASC
                        ;

We were tasked with creating a query that included a sub query with a formatted output from a selection of databases. I chose to try and find who got sent off during the Euro 2016 football tournament. Included are the table structure, and my SQL query.

We used PostgreSQL for the challenge, and so i basically selected the distinct player names (since i only wanted to find out if they got sent off at any point, not using distinct would tell me if they got sent off more than once), and some other information i thought was important from the player_mast table. The subquery then checks the player_booked table, and returns all the player id's of the players who got sent off, and then the where condition matches the player id's returned, with player id's in the player_mast table to give me the information on everyone who got sent off.

Laravel CRUD functionality (create)


                        public function store(companies $company)
                        {
                    
                            if(request('logo') == null) {
                    
                                companies::insert([$this->companyRules()]);
                    
                            } else {
                    
                                companies::insert([
                                    array_merge($this->companyRules(), [
                                        'logo' => request()->file('logo')->store('public')
                                    ]
                                )]);
                            }
                    
                            return redirect('/companies');
                        }

                        protected function companyRules()
                        {
                            return request()->validate([
                                'name' => 'required|max:255',
                                'email' => 'nullable|email',
                                'logo' => 'nullable|image|dimensions:min_width=100,min_height=100',
                                'website' => 'nullable|url'
                            ]);
                        }

We were tasked with creating an admin site to store a list of companies and their employees using laravel, and this is the create function from it.

When you submit the form to add a company, the store method checks to see if there's a logo being inserted (this check is required since the logo field is nullable and the store method cannot be null) and then calls the company rules validation method.

After applying the specific validation rules we need, the store method then uses laravel's insert method to create a behind the scenes sql query to insert the validated input. If there is a logo being inserted, then we call the store method and create a symbolic link between our storage folder, and our public folder, so we can store and display the logo in our site.

I made a game built entirely in html/css/javascript based on a space invaders and battleships crossover idea i had.

Basically, you control a small ship searching for invisible enemy ships aiming to shoot them down. Your ship can move left and right across a 21 column grid, and fire upwards to hit enemies (the space invaders part). The enemy ships are invisible, and it's your job to shoot the correct column to destroy them (the battleships part). You have ten shots to find three enemy ships, and once your ten shots are fired, if you haven't destroyed the enemy ships, then it's game over.

The code above is for randomising the enemy ship positions at the start of the game. The arrays store the positions based on css absolute positioning values, and the math.random functions pull a random number from those array of positions, and apply them to the left and bottom positions of the enemy ships.

As of writing, there are some bugs with the game, and some improvements i'd like to make (graphical things, and i'd like to add a leaderboard of some kind with php) but it's in a playable state, so, as the games industry likes to say, those aren't bugs, they're "features".
